# rphys

`rphys` is a Python package for remote physiological measurement research, providing reusable components for domain-specific datasets, preprocessing, models, training, evaluation, and analysis.

This repository is being rebuilt as a clean base library for reproducible research workflows. The current focus is scaffold planning: converting project context into a roadmap, turning accepted roadmap work packages into implementation master plans, and running managed multi-agent implementation in isolated git worktrees.
