# rphys

`rphys` is a Python package for remote physiological measurement research, providing reusable components for domain-specific datasets, preprocessing, models, training, evaluation, and analysis.

This repository is being rebuilt as a clean base library for reproducible
research workflows. The current architecture and implementation direction are
captured in the canonical roadmap.

Start with:

- `AGENTS.md` for Codex and contributor instructions.
- `docs/rphys_implementation_roadmap.md` for the current implementation
  roadmap and public architecture policy.
