# rphys

`rphys` is a Python package for remote physiological measurement research, providing reusable components for domain-specific datasets, preprocessing, models, training, evaluation, and analysis.

This repository is being rebuilt as a clean base library for reproducible research workflows. The current focus is workflow scaffolding for carefully auditing a legacy codebase, designing modern research components, implementing them in isolated git worktrees, and reviewing them for both code quality and scientific correctness.

Start with:

- `AGENTS.md` for Codex and contributor instructions.
- `docs/workflows/README.md` for the multi-agent porting workflow.
- `docs/roadmap/index.md` for the canonical roadmap.
