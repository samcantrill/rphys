# Codex Prompts

This directory contains canonical reusable prompts for `rphys` workflow actions.

- `.codex/workflows/` contains short user-facing workflow entrypoints.
- `.codex/prompts/` contains role/action behavior for drafting, refining, reviewing, implementing, and preparing artifacts.
- `.codex/templates/` contains durable Markdown artifact schemas.
- `.codex/agents/` contains model, sandbox, and role authority.

Prompt filenames use an artifact/action shape, for example `stage-1-planning-notes-facilitate.md`, `master-plan-review.md`, and `phase-pr-body-draft.md`.
